Historia da WEB

A WEB é um serviço criado pelo Tim Bernes Lee, 1990. Que funcina sobre a internet, que é uma rede de computadores, utiliza a arquitetura Cliente - Servidor

Para isso ele definiu a estrutura de funcionamento web, cliente servidor e criou a linguagem HTML, para exibição de documentos e o protocolo HTTP para comunicação entre a aplicação cliente e servidor

Para cuidar dos padrões da web e prorpiedade da tecnologia ele criou a fundação w3c World Web Consorcium

A referência das tags HTML pode ser encontrado no site da w3c
