Comandos GIT

  • git init- criar um repositorio;
  • git add [file] :adiciona um arquivo no stage;
  • git status : exibe o estado do rtepositorio;
  • commit -m "msg" : cria um commit;
  • log : exibe o commits do repositorio, da branch atual;
  • git checkout referencia ; Recupera o conteudo da referencia para WK;
  • git remote add nome url :definir um repositorio remoto;
  • git push url branch : envia a branch paar repositorio remoto;
